01
Python & Math Foundations
4–6 wks Beginner

Build the mathematical and programming bedrock. Every model you'll write later is built on these primitives.

🐍Python: NumPy, Pandas, Matplotlib
∫Linear algebra — vectors, matrices, dot products
📈Calculus — derivatives, chain rule, gradients
🎲Statistics — probability, distributions, Bayes' theorem
🖼️Image basics — pixels, channels, histograms
Resources fast.ai Practical DL 3Blue1Brown — Linear Algebra CS50P
🏁 Milestone: Load, visualize & normalize a JPEG in NumPy without any ML library
02
Classical ML & scikit-learn
3–4 wks Beginner+

Understand how learning algorithms work conceptually before touching neural networks. Many baselines are classical models.

📐Logistic Regression, SVM, k-NN
🌲Decision Trees, Random Forests, XGBoost
⚖️Train/val/test splits, cross-validation
📊Metrics: accuracy, precision, recall, AUC-ROC
🔧Overfitting, regularization, bias-variance
Resources scikit-learn docs Hands-On ML (Géron) Kaggle Learn ML
🏁 Milestone: Classify skin lesion images using HOG features + SVM; report AUC score
03
Deep Learning & CNNs
6–8 wks Intermediate

The core engine of skin lesion detection. Convolutional Neural Networks learn hierarchical visual features automatically — no hand-crafted descriptors needed.

🧠Perceptrons, activation functions, backprop
🔲Convolution, pooling, stride, padding
🏗️Architectures: ResNet, EfficientNet, DenseNet
⚙️Optimizers: Adam, SGD, learning rate schedules
🔥PyTorch & TensorFlow/Keras fundamentals
🛡️Batch norm, dropout, weight decay
Resources Deep Learning (Goodfellow) CS231n Stanford PyTorch Tutorials
🏁 Milestone: Train a CNN from scratch on CIFAR-10 with >85% accuracy
04
Medical Imaging & Dermoscopy Domain
3–4 wks Intermediate

Medical images are fundamentally different from ImageNet photos. You need domain knowledge before working on clinical data.

🔬Dermoscopy vs standard photography
🗂️HAM10000 & ISIC datasets — 7 lesion classes
🏷️ABCD rule: Asymmetry, Border, Color, Diameter
⚖️Class imbalance (melanoma is rare)
🛡️Ethics, FDA regulations, clinical workflows
🎨Medical augmentation: color jitter, flips, hair removal
Resources ISIC Challenge papers HAM10000 paper (Tschandl 2018) MICCAI proceedings
🏁 Milestone: Perform full EDA on HAM10000; document class distribution and visual patterns
05
Transfer Learning & Advanced Architectures
6–8 wks Advanced

No model in production is trained from scratch on skin lesions alone. Transfer learning from ImageNet pretrained weights is the industry standard.

♻️Fine-tuning vs feature extraction
⚡EfficientNet-B4/B7 — ISIC leaderboard standard
🤖Vision Transformers (ViT, Swin Transformer)
🎭Data augmentation: Albumentations library
🧪Weighted loss, focal loss for class imbalance
🗳️Test-time augmentation (TTA) & ensembles
🔍Grad-CAM, SHAP — model explainability
Resources timm library (HuggingFace) Albumentations docs ISIC 2018–2020 winning solutions
🏁 Milestone: Fine-tune EfficientNet-B4 on HAM10000 — achieve AUC ≥ 0.90
06
MLOps, Evaluation & Validation
4–5 wks Advanced

A model that isn't rigorously evaluated and tracked is a liability, not an asset — especially in healthcare.

📏Clinical metrics: sensitivity, specificity, PPV, NPV
📉ROC curves, calibration curves, confusion matrices
📋Experiment tracking: MLflow / Weights & Biases
🔄Cross-validation strategies for medical data
🏗️Model versioning & reproducibility
⚠️Distribution shift & domain generalization
Resources W&B documentation MLflow tutorials scikit-learn metrics guide
🏁 Milestone: Full evaluation report — confusion matrix, ROC, Grad-CAM visualizations, logged in W&B
07
Deployment & Production
3–4 wks Advanced

Ship a real, usable system. A working deployment is the clearest proof of competence and is essential for portfolio impact.

🚀ONNX / TorchScript model export
🌐FastAPI REST endpoint for inference
🖥️Gradio / Streamlit demo UI
🐳Dockerize and deploy on HuggingFace Spaces or AWS
⚡ONNX quantization & latency optimization
📱(Optional) TFLite / CoreML for mobile
Resources FastAPI docs HuggingFace Spaces ONNX Runtime
🏁 Milestone: Public live demo — user uploads a lesion image and receives a prediction with heatmap overlay
